# **App Name**: AcademiaLite

## Core Features:

- User Role Authentication: Secure user authentication based on predefined roles (Super Admin, Teacher, Student).
- Dashboard Overview: Displays key metrics such as Total Students and Recent Activities.
- Student Profile Viewer: Allows the current user to view student profiles.
- Teacher Profile Viewer: Allows the current user to view teacher profiles.
- Profile Editor: Users can modify basic profile information like password and contact information.
- Intelligent Announcement Summarizer: Uses a generative AI tool to condense long academy announcements into a short, user-friendly summaries.

## Style Guidelines:

- Primary color: Soft blue (#72BCD4), inspired by a sense of calm and focus conducive to education.
- Background color: Light gray (#F0F4F7), a desaturated version of the primary, offering a clean and unobtrusive backdrop.
- Accent color: Muted green (#80CBC4), analogous to blue but with enough contrast to draw attention to important elements.
- Font: 'PT Sans', a humanist sans-serif that provides a modern and readable feel, suitable for both headlines and body text.
- Use minimalist line icons to represent different sections and actions within the app, ensuring clarity without visual clutter.
- Implement a clean and responsive layout with clear content sections to improve usability on all devices.
